# pdfTranslator
![](https://img.shields.io/badge/Macos-m1pro-orange) ![](https://img.shields.io/badge/python-3.8.13-blue) ![](https://img.shields.io/badge/jquery-3.6.0-red) ![](https://img.shields.io/badge/bootstrap-5.0.0-green)\
This application is to automatically translate pdf files such as research paper into Mandarin Chinese and adjust the output doc file according to the need of users

## Prerequisites
### 1. Virtual Environment ###
#### 1) Create a virtual environment ####
`python3 -m venv myenv`

##### 2) install all required package in this application
`pip install -r requirements.txt`

## Run Server
Go to the path of app and run :
`python app.py`

## Future Improvement
